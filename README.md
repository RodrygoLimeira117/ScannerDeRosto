# Face Emotion Detector

Este é um projeto Python que detecta rostos e emoções em tempo real (via webcam) ou em imagens estáticas.

## Requisitos

- Python 3.8+
- OpenCV
- dlib
- deepface

## Uso

```bash
python main.py
```

Escolha entre:
1. Usar a webcam em tempo real
2. Carregar uma imagem para detecção
